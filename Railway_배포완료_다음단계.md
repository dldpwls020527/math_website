## 🌐 URL 확인하기

### 방법 1: 서비스 카드에서 확인 (가장 쉬움)

1. **서비스 카드 클릭**
   - 화면 중앙의 `web` 서비스 카드를 클릭
   - 또는 카드에 표시된 URL을 확인
   - 예: `web-production-563ea.up.railway.app`

2. **URL 복사**
   - URL을 클릭하거나 복사
   - `https://`가 앞에 붙어야 합니다

### 방법 2: Settings 탭에서 확인

1. 상단 **"Settings"** 탭 클릭
2. **"Domains"** 섹션 찾기
3. URL 확인 및 복사
   - 예: `https://web-production-563ea.up.railway.app`

### 방법 3: 서비스 상세 페이지

1. `web` 서비스 카드 클릭
2. 상세 페이지에서 URL 확인
3. **"Generate Domain"** 버튼이 있다면 클릭하여 도메인 생성
