# ⚡ 빠른 시작: GitHub에 코드 업로드하기

GitHub 사용자명: **dldpwls020527**

## 🎯 가장 쉬운 방법 (5분)

### 1단계: GitHub Desktop 설치 (2분)
1. 브라우저에서 https://desktop.github.com/ 열기
2. "Download for Windows" 클릭
3. 다운로드된 파일 실행 및 설치
4. 설치 완료 후 GitHub Desktop 실행

### 2단계: GitHub 계정 로그인 (1분)
1. GitHub Desktop에서 "Sign in to GitHub.com" 클릭
2. 사용자명: **dldpwls020527**
3. 비밀번호 입력 후 로그인

### 3단계: 저장소 생성 및 업로드 (2분)
1. GitHub Desktop에서 **"File"** → **"Add Local Repository"** 클릭
2. **"Choose..."** 버튼 클릭
3. 다음 폴더 선택:
   ```
   C:\Users\dldpw\Desktop\math
   ```
4. **"Add repository"** 클릭
5. 좌측 하단에 변경사항 확인됨
6. 좌측 하단 "Summary"에 메시지 입력: `Initial commit: 수학 공식 웹사이트`
7. **"Commit to main"** 버튼 클릭
8. 상단 메뉴 **"Repository"** → **"Publish repository"** 클릭
   - 또는 좌측 상단에 "Publish repository" 버튼이 보이면 클릭
9. Repository name: `math-website` (또는 원하는 이름)
10. **"Keep this code private" 체크 해제** (Public으로 설정)
11. **"Publish Repository"** 버튼 클릭

### 4단계: 완료! ✅
- 업로드 완료!
- 브라우저에서 확인: https://github.com/dldpwls020527/math-website

## 🔄 코드 수정 후 재업로드

1. 파일 수정
2. GitHub Desktop 열기
3. 변경사항 자동 감지됨
4. 좌측 하단에 메시지 입력 (예: "Update formula")
5. **"Commit to main"** 클릭
6. 상단 **"Push origin"** 버튼 클릭 (또는 "Repository" → "Push")
