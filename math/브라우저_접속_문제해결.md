# 🔧 브라우저 접속 문제 해결 가이드

브라우저에서 웹사이트가 열리지 않는 문제를 해결하는 방법입니다.

---

## 📋 문제 진단

### 1단계: 서버가 실행 중인지 확인

서버가 실행되지 않으면 브라우저에서 접속할 수 없습니다.

**확인 방법:**
```powershell
netstat -ano | findstr :5000
```
- 결과가 없으면 → 서버가 실행되지 않음
- 결과가 있으면 → 서버가 실행 중

---

## 🔧 해결 방법

### 방법 1: 서버 실행 (가장 흔한 문제)

1. **PowerShell 또는 터미널 열기**
2. **프로젝트 폴더로 이동**
   ```powershell
   cd C:\Users\dldpw\Desktop\math
   ```

3. **서버 실행**
   ```powershell
   python app.py
   ```

4. **성공 메시지 확인**
   ```
   * Running on http://0.0.0.0:5000
   ```

5. **브라우저에서 접속**
   - `http://localhost:5000`
   - 또는 `http://127.0.0.1:5000`

---

### 방법 2: 포트가 다른 경우

다른 프로그램이 포트 5000을 사용 중일 수 있습니다.

**포트 변경하여 실행:**
```powershell
$env:PORT=8080
python app.py
```

그 다음 브라우저에서:
- `http://localhost:8080`

---

### 방법 3: 오류가 발생하는 경우

서버 실행 시 오류 메시지가 나타나면:

1. **오류 메시지 확인**
   - 어떤 오류인지 메시지 읽기

2. **일반적인 오류:**

   **오류: "ModuleNotFoundError"**
   ```powershell
   pip install -r requirements.txt
   ```

   **오류: "Port already in use"**
   - 다른 포트 사용 또는 해당 포트 사용 중인 프로그램 종료

   **오류: "FileNotFoundError"**
   - 필요한 파일이 있는지 확인
   - `data/formulas.json` 파일 확인

## 🌐 Railway 배포된 사이트 접속 문제

### 문제: Railway URL이 작동하지 않는 경우

1. **Railway 대시보드 확인**
   - https://railway.app 접속
   - 프로젝트 상태 확인
   - "Active" 상태인지 확인

2. **URL 확인**
   - 올바른 URL인지 확인
   - `https://`로 시작하는지 확인

3. **재배포 확인**
   - 최근 배포가 실패했는지 확인
   - 로그에서 오류 확인
