# 🖼️ 이미지를 웹사이트에 표시하는 방법

현재 상황과 이미지를 보이게 하는 방법을 안내합니다.

---

## 📋 현재 상태

### ✅ 실제 이미지 파일들
- `static/images/근의공식.png`
- `static/images/원의 넓이.png`
- `static/images/이차함수.png`
- `static/images/피타고라스.png`

### ⚠️ 문제점
현재 `formulas.json`에서 이미지 경로가 실제 파일명과 일치하지 않을 수 있습니다.

---

## 🔧 이미지를 보이게 하는 방법

### 방법 1: 이미지 관리 페이지 사용 (추천) ⭐

1. **이미지 관리 페이지 접속**
   - `http://localhost:5000/admin/images`
   - 또는 Railway URL: `https://your-url.railway.app/admin/images`

2. **이미지 확인**
   - 업로드된 이미지 목록에서 이미지 확인
   - 이미 업로드되어 있다면 그대로 사용

3. **공식에 이미지 연결**
   - 이미지 목록에서 원하는 이미지의 "선택" 버튼 클릭
   - 연결할 공식 찾기
   - "이미지 연결" 버튼 클릭

4. **완료!**
   - 변경사항이 자동으로 저장됨
   - 웹사이트에서 이미지 확인

---

### 방법 2: formulas.json 직접 수정

`formulas.json` 파일에서 이미지 경로를 직접 수정:

```json
{
  "id": 1,
  "name": "이차방정식의 근의 공식",
  "image": "/static/images/근의공식.png"
}
```

---

## ✅ 이미지가 보이는지 확인

### 확인 방법

1. **웹사이트 접속**
   - 로컬: `http://localhost:5000`
   - Railway: `https://your-url.railway.app`

2. **공식 상세 페이지 확인**
   - 공식 목록에서 공식 클릭
   - 이미지가 있는 공식은 "📷 시각적 설명" 섹션에 이미지 표시

3. **이미지가 보이면 성공!** ✅

---

## 🚀 Railway 배포 후

Railway에 배포한 후에도 이미지가 보이려면:

1. **이미지 파일이 GitHub에 업로드되어야 함**
   - `static/images/` 폴더의 모든 이미지 파일이 GitHub에 있어야 함
   - GitHub Desktop에서 `static/images/` 폴더 확인

2. **formulas.json이 업데이트되어야 함**
   - 이미지 경로가 올바르게 설정되어야 함
   - 이미지 관리 페이지에서 연결하면 자동 업데이트됨

---

## 💡 팁

1. **이미지 파일명**
   - 한글 파일명도 사용 가능
   - 공백 포함 가능 (예: "원의 넓이.png")

2. **이미지 형식**
   - PNG, JPG, JPEG, GIF, WEBP 지원

3. **이미지 크기**
   - 너무 큰 이미지는 로딩이 느릴 수 있음
   - 최적 크기: 800x600 픽셀 정도

---

## 🆘 이미지가 안 보이는 경우

1. **파일 경로 확인**
   - `formulas.json`의 이미지 경로 확인
   - 실제 파일 경로와 일치하는지 확인

2. **파일 존재 확인**
   - `static/images/` 폴더에 파일이 있는지 확인

3. **브라우저 콘솔 확인**
   - F12 키 누르기
   - Console 탭에서 오류 메시지 확인

4. **이미지 관리 페이지 사용**
   - `/admin/images`에서 이미지를 다시 연결

---

**이제 이미지 관리 페이지를 사용하여 이미지를 연결하시면 웹사이트에서 이미지가 보입니다!** 🎉
