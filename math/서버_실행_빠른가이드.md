# 🚀 서버 실행 빠른 가이드

브라우저 접속이 안 되면 서버를 실행해야 합니다!

---

## ⚡ 빠른 실행 (3단계)

### 1단계: PowerShell 열기
- Windows 키 + X
- "Windows PowerShell" 또는 "터미널" 선택

### 2단계: 폴더로 이동
```powershell
cd C:\Users\dldpw\Desktop\math
```

### 3단계: 서버 실행
```powershell
python app.py
```

---

## ✅ 성공 확인

서버가 실행되면 다음과 같은 메시지가 나타납니다:

```
 * Running on http://0.0.0.0:5000
 * Debug mode: off
```

이제 브라우저에서 접속 가능합니다!

---

## 🌐 브라우저에서 접속

서버가 실행 중이면:
- `http://localhost:5000` 
- 또는 `http://127.0.0.1:5000`

---

## 🆘 문제 해결

### "python을 찾을 수 없습니다" 오류

**해결:**
```powershell
# Python 경로 확인
where python

# 또는 py 사용
py app.py
```

### 포트가 이미 사용 중입니다

**해결:**
```powershell
# 다른 포트 사용
$env:PORT=8080
python app.py

# 그 다음 브라우저에서
http://localhost:8080
```

### 패키지 오류

**해결:**
```powershell
pip install -r requirements.txt
```

   - PowerShell 창을 닫으면 서버가 중지됩니다
   - 서버를 중지하려면: Ctrl + C

   - 서버가 실행 중인 터미널은 그대로 두고
